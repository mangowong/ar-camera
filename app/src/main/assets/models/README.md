# 3D Models Directory

This directory contains 3D models for the AR Camera app.

## Available Models

1. **cesium_man.glb** - CesiumMan character model (480KB)
   - Main character for AR scene
   - High-quality humanoid model with animations

2. **animated_box.glb** - Animated box model (12KB)
   - Sample animated model for testing

## Model Sources

- CesiumMan: [Khronos glTF Sample Models](https://github.com/KhronosGroup/glTF-Sample-Models)
- Animated Box: [Khronos glTF Sample Models](https://github.com/KhronosGroup/glTF-Sample-Models)
